1) 개발 환경  
 - Language: Java  
 - Platform: Android  
 - Tool: eclipse, SVN  
 
2) 프로젝트 개요  
 - 한전 내부에서 사용되는 예방점검/정밀보수/공사관리 안드로이드 하이브리드 형 앱 제작  

3) 프로젝트 구현 기능  
 - Google Map 연동 기능  
 - 웹 페이지 연동 기능 (Hybrid)  
 - Fragment 기반 UI 분할 처리 기능  
 - 이미지 로드 및 REST API 연동 기능  
 - 캘린더(만년달력) 및 일정 관리 기능
